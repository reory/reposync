# 🚀 RepoSync
A high-performance developer portfolio built with **Python 3.13** and **Reflex**.

## ✨ Features
- **Live GitHub Sync**: Pulls real-time data from the GitHub API.
- **Instant Search**: Reactive filtering as you type.
- **Modern UI**: Dark mode with an "Iris" color theme.
- **No JavaScript**: Built 100% in Python.😁

## 🛠️ Installation
1. Clone the repo.
2. Create a venv: `python -m venv venv`
3. Install: `pip install reflex requests`
4. Run: `reflex init`
5. Run: `reflex run`
